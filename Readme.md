# 基于学生画像的智能学业建议系统

本项目旨在通过构建多维度的学生画像，结合大语言模型技术，为学生提供个性化的学业建议与指导。系统主要包含以下四大核心模块：

## 1. 用户信息管理模块
- **技术栈**：Flask + MySQL（后端），Vue + Element Plus + Vue Router（前端）。
- **功能描述**：实现了用户的注册、登录及角色权限管理。
- **用户角色**：支持 **学生**、**教师** 和 **管理员** 三类用户的身份认证与信息维护需求。

## 2. 数据采集与预处理模块
- **数据来源**：支持导入 CSV/Excel 文件进行数据采集。
- **数据处理**：使用 `pandas` 库进行数据清洗，包括缺失值填充、异常值剔除、重复记录合并及格式标准化等操作。
- **数据存储**：最终以 **学号** 为唯一标识，将结构化数据统一存入 MySQL 数据库，为后续画像构建提供基础数据支撑。

## 3. 学生画像核心构建模块
- **构建策略**：采用“规则驱动为主、聚类为辅”的策略。
    - **基础标签**：通过 JSON 配置生成成绩等级、偏科情况等基础标签。
    - **复合特征**：使用 K-Means 聚类算法（基于 `scikit-learn`）划分典型的学习风格等复合特征。
- **可视化展示**：
    - 前端基于 Vue 与 ECharts，以 **卡片**、**雷达图** 等形式直观展示个体画像。
    - 支持按 **专业**、**年级** 筛选查看群体统计图表，实现直观、可交互的学情呈现。
- **数据更新**：标签定时更新并存入 MySQL。

## 4. 智能学业建议模块
- **核心技术**：调用 **通义千问 (Qwen)** 大模型开放 API。
- **工作流程**：
    1. 将学生画像标签转换为自然语言 Prompt。
    2. 后端构造请求发送给大模型，生成面向不同角色的学业建议。
    3. 返回文本结果并在前端进行展示。
